# Flows — lifecycle и шаблоны

Каталог `memory-bank/flows/` содержит reusable process-layer: lifecycle rules, taxonomy стабильных идентификаторов и governed templates для feature-документации.

## Навигация

- [Feature Flow](feature-flow.md) — lifecycle от draft до closure, transition gates и стабильные ID (`REQ-*`, `SC-*`, `CHK-*`, `STEP-*` и т.д.). Читать перед созданием нового feature package с FT-006+.
- [Templates](templates/index.md) — эталонные шаблоны feature-документов: `README.md`, `short.md`, `large.md`, `implementation-plan.md`.
